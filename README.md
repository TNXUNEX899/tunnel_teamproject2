Smart Tunnel Flood Alert System
โปรเจกต์นี้เป็นการผสานการทำงานระหว่าง IoT, Firebase, Flutter เพื่อเฝ้าระวังและแจ้งเตือนผู้ใช้รถใช้ถนนให้รอดพ้นจากอันตรายของน้ำท่วมฉับพลันในอุโมงค์และทางลอด

Smart Map & Pins:แผนที่พร้อมหมุดระบุตำแหน่งอุโมงค์ที่เปลี่ยนสีอัตโนมัติแบบเรียลไทม์ (เขียว=ปลอดภัย, เหลือง=เฝ้าระวัง, แดง=อันตราย)
Real-time Dashboard:หน้าต่างแสดงสถานะระดับน้ำปัจจุบันในรูปแบบเปอร์เซ็นต์ พร้อมกราฟแท่ง (Bar Chart) แสดงประวัติระดับน้ำย้อนหลัง 6 ค่าล่าสุด
Weather Integration:ดึงข้อมูลสภาพอากาศแบบเรียลไทม์จาก Weather API มาแสดงผล (อุณหภูมิ, สภาพอากาศ, ความเร็วลม, ความชื้นสัมพัทธ์, UV Index) เพื่อประเมินสถานการณ์ฝนตก
Auto-Cleanup Database:มีระบบจัดการข้อมูลประวัติระดับน้ำแบบ FIFO ลบข้อมูลเก่าทิ้งอัตโนมัติเพื่อให้กราฟและแอปพลิเคชันทำงานได้รวดเร็ว
